# TimeZoneColors Indicator

## Overview
The TimeZoneColors indicator allows traders to visualize different time zones on their charts using color-coded bands. This helps in identifying key market opening and closing times across different regions.

## Features
- **Color-coded Time Zones:** Easily differentiate between various market hours.
- **Customizable Colors:** Adjust the colors of each time zone to suit your preferences.
- **Session Highlighting:** Automatically highlight the current trading session for better insights.

## Installation
1. Clone the repository or download the ZIP file.
2. Navigate to the Indicators folder.
3. Copy the `TimeZoneColors` folder to your NinjaTrader 8 indicators directory.

## Configuration
- Open NinjaTrader 8.
- Go to the 'Tools' menu and select 'NinjaScript Editor'.
- Locate the TimeZoneColors indicator in the editor.
- Modify the settings as required, including color options for different time zones.

## Usage Examples
To use the TimeZoneColors indicator:
1. Add the indicator to your desired chart.
2. Configure the colors for each time zone as per your trading strategy.
3. Observe the color-coded zones to make timely trading decisions.

## Troubleshooting
- **Indicator Not Appearing:** Ensure that the indicator is correctly installed in the NinjaTrader indicators directory.
- **Colors Not Displaying:** Double-check the configuration settings and ensure the color codes are valid.
- **Performance Issues:** If the indicator slows down your chart, consider optimizing the time zone settings.
