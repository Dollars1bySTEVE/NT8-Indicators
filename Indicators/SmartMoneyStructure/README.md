# SmartMoneyStructure Indicator

## Features
- Identifies key market structure levels based on smart money principles.
- Visual representation of support and resistance levels.
- Alerts for potential reversals and breakouts.

## Installation
1. Clone the repository: `git clone https://github.com/Dollars1bySTEVE/NT8-Indicators.git`
2. Navigate to the Indicators folder.
3. Ensure all dependencies are installed.

## Configuration
- Set up the indicator through the settings menu in NinjaTrader.
- Adjust parameters such as sensitivity and alert types to fit your trading strategy.

## Usage Examples
- To observe market structure, apply the indicator to your chart and adjust settings as necessary.
- Use the alerts for better decision-making during trading sessions.

## Troubleshooting
- If the indicator is not displaying correctly, check your settings and ensure that all required dependencies are in place.
- Consult the documentation or seek help in the support forum if issues persist.
