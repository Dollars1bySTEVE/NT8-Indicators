# D1ADR Indicator

## Overview
The D1ADR (Daily Average Daily Range) indicator is a powerful tool designed for traders to analyze market volatility on a daily basis. It calculates the average daily range over a specified period to help make informed trading decisions.

## Features
- **Volatility Measurement**: Measures average daily volatility over a configurable period.
- **Customizable Period**: Users can adjust the look-back period to fit their trading strategy.
- **Visual Representation**: Displays average range as a line on the chart for easy analysis.

## Installation
To install the D1ADR indicator, follow these steps:
1. **Download** the indicator file from the repository.
2. **Open** your trading software (e.g., NinjaTrader, MetaTrader).
3. **Import the Indicator**:
   - For NinjaTrader: Go to `File` > `Import` > `NinjaScript Add-On...` and select the downloaded file.
   - For MetaTrader: Copy the `.ex4` file to the `Indicators` folder in the MetaTrader directory.

## Configuration
After installation, you can configure the D1ADR indicator by:
1. Adding the indicator to your chart.
2. Accessing the settings menu to adjust:
   - Look-back period (default: 14 days)
   - Color and style of the line on the chart
   - Alerts for when price exceeds the average range

## Usage Examples
- **Day Trading**: Utilize the D1ADR to determine potential entry and exit points based on current volatility.
- **Swing Trading**: Assess the average range to identify potential breakout points and set profit targets.

## Troubleshooting
- **No Data Showing**: Ensure that the indicator is correctly installed and applied to a chart with sufficient historical data.
- **Errors during Installation**: Verify that you are using a compatible trading platform and the correct versions of the indicator.
- **Performance Issues**: If the indicator slows down your platform, reduce the look-back period or check for conflicts with other indicators.
