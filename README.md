## 5. D1ADR

### What It Does
| Feature              | Description  |
|----------------------|--------------|
| Daily Average Range  | Measures the average range of the daily price movements. |
| Customizable Lines    | Allows for selection of lines to display the range. |

### ADR Parameters Settings
- **ADR Length**: (default: 14) Number of days to calculate the average range.
- **ADR Multiplier**: (default: 1) Multiplier for the calculated average range.

### ADR Lines Customization
- **Show Lines**: (default: true) Displays the starting point and ending point of the ADR.
- **Line Color**: (default: #ff0000) Color for the ADR lines.

### ADR Fill Options
- **Show Fill**: (default: false) Option to fill the area between the lines.
- **Fill Color**: (default: rgba(255, 0, 0, 0.1)) Color used for filling the area.

### Daily Open Settings
- **Show Daily Open**: (default: true) Displays the daily opening price line.

### Session Opens
- **Asia Session**: (default: true) Displays a marker for the Asia session open.
- **London Session**: (default: true) Displays a marker for the London session open.
- **NY Session**: (default: true) Displays a marker for the NY session open.

### Weekly Range (AWR) Settings
- **Show AWR**: (default: true) Displays the average weekly range.
- **AWR Length**: (default: 14) Number of weeks for calculating the average.

### Daily Pivots
- **Show Daily Pivots**: (default: true) Option to display daily pivot points.

### Weekly Pivots
- **Show Weekly Pivots**: (default: true) Option to display weekly pivot points.

### Labels & Info Customization
- **Show Labels**: (default: true) Displays labels for ADR and session opens.
- **Label Color**: (default: #000000) Color of the labels.
